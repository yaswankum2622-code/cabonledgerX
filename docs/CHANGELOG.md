# Changelog

This changelog tracks the major repository milestones. The project evolved in phase-based increments rather than semantic-versioned public releases.

## 2026-04-23

### Repository Packaging And Presentation

- upgraded the dashboard into a premium single-page product surface
- added FastAPI read-only service endpoints over processed parquet outputs
- added scoring reconciliation and audit artifacts
- organized documentation into a numbered architecture and interview-prep system

### Analytical Core

- added probabilistic commitment-miss scoring with calibration comparison
- added statistical forecasting, backtesting, and evaluation artifacts
- added historical annual reconstruction from 2015 through 2024
- added activity-based emissions calculator and company activity input generator

### Product Intelligence Layers

- added contradiction flags and rule-based scoring
- added intervention simulator, MAC ranking, and intervention intelligence
- added evidence-pack generation for board, investor, and lender audiences

### Core Pipeline

- added canonical processed tables from eGRID, DEFRA, and SBTi
- added interim extraction and raw workbook profiling
- established project structure, settings, path helpers, bootstrap validation, scripts, and smoke tests

## Repository Maturity Summary

The repository currently includes:

- ingestion and profiling
- canonical data modeling
- calculator and history layers
- deterministic and statistical forecasts
- heuristic and probabilistic scoring
- reconciliation logic
- intervention intelligence
- dashboard and API surfaces
- documentation, ADRs, and interview-prep support

## Not Yet A Formal Release

The project does not yet use:

- tagged release notes
- semantic versioning
- packaged deployment artifacts
- database-backed serving

Those are appropriate next steps after public-repo cleanup and hosting decisions.
